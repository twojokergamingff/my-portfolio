# Twojokergaming Portfolio

A modern, interactive portfolio website showcasing my work as a Roblox developer, 3D modeler, and community manager.

## Features

- 🎨 **Modern Design** - Sleek, Roblox-inspired aesthetic with cyan/blue color scheme
- 📱 **Responsive** - Fully responsive design that works on all devices
- ⚡ **Fast Performance** - Built with React + Vite for optimal speed
- 🎯 **Interactive Elements** - Smooth animations and hover effects
- 📊 **Project Showcase** - Display of featured projects and accomplishments

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Next-gen build tool
- **CSS3** - Modern styling with animations

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/twojokergamingff/my-portfolio.git

# Navigate to the project
cd my-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

The portfolio will open at `http://localhost:3000`

### Build

```bash
# Create a production build
npm run build

# Preview the production build
npm run preview
```

## Features

### Sections

1. **Navbar** - Navigation menu with smooth scrolling
2. **Hero** - Eye-catching introduction with animated cubes
3. **Skills** - Showcase of expertise in various areas
4. **Projects** - Featured projects with descriptions and tags
5. **Contact** - Links to social media platforms
6. **Footer** - Site information and copyright

## Customization

To customize the portfolio:

1. Edit component content in `src/components/`
2. Modify colors in CSS files (search for `#00d9ff` for the main accent color)
3. Update project information in `Projects.tsx`
4. Modify contact links in `Contact.tsx`
